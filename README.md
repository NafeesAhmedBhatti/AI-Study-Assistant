# 🚀 Overview

SmartStudy AI is an intelligent study platform designed to help students learn faster and retain more information.

Instead of manually creating notes, flashcards, quizzes, and study plans, students can upload learning material and let AI generate everything automatically.

The platform combines productivity tools, AI tutoring, and personalized learning workflows into one modern application.

# ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/NafeesAhmedBhatti/AI-Study-Assistant.git
```

Move into project:

```bash
cd AI-Study-Assistant
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

# 🏗 Build Project

```bash
npm run build
```

# 🚀 Deploy To GitHub Pages

```bash
npm run build
```

Copy build output:

```bash
dist → docs
```

Commit changes:

```bash
git add .
git commit -m "Deploy latest build"
git push origin main
```
